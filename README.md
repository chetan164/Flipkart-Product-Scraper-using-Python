# 🛒 Flipkart Product Scraper using Python

## 📌 Project Overview

This project is a Python-based web scraping tool that extracts product data such as names, prices, and ratings from Flipkart. It demonstrates practical skills in data extraction, cleaning, and automation using real-world e-commerce data.

---

## 🚀 Features

* 🔍 Scrapes product details (Name, Price, Ratings)
* ⚡ Fast and automated data extraction
* 🧹 Data cleaning and preprocessing using Pandas
* 📊 Structured data storage for analysis
* 💾 Export data to CSV/Excel format

---

## 🛠️ Tech Stack

* **Python** 🐍
* **BeautifulSoup** 🌐
* **Requests** 📡
* **Pandas** 📊

---

## 📂 Project Workflow

1. Send HTTP request to Flipkart website
2. Parse HTML content using BeautifulSoup
3. Extract required product details
4. Clean and structure data using Pandas
5. Export final dataset for analysis

---

## 📈 Output

* Extracted 100+ product records
* Clean and structured dataset ready for analysis
